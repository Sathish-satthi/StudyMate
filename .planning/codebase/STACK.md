# Tech Stack

## Frontend
- **Framework**: React 19 (Vite)
- **Routing**: react-router-dom v7
- **Styling**: Vanilla CSS (CSS Modules/Global)
- **API Client**: Axios

## Backend
- **Runtime**: Node.js
- **Framework**: Express 5
- **Database**: MongoDB (Mongoose 9)
- **Authentication**: JWT + Bcryptjs

## Infrastructure
- **Development**: Nodemon (server), Vite (client)
- **Environment**: Dotenv for configuration

## Versions (Current)
- `react`: ^19.2.4
- `express`: ^5.2.1
- `mongoose`: ^9.4.1
- `vite`: ^8.0.4
