# Project: StudyMate

## What This Is
A premium, minimalist study platform designed for classmates to share and access subject notes and important exam questions efficiently. The focus is on a clean, distraction-free interface that provides high utility with a "premium simple" feel.

## Core Value
Empower students with a centralized, easy-to-use repository of collective knowledge to improve exam preparation and academic performance.

## Context
- **Stage**: Early Development (Auth scaffolding present)
- **Tech Stack**: MERN (MongoDB, Express, React, Node.js) with Vite.
- **UI Vision**: "Premium Simple" — Clean lines, high-quality typography (e.g., Inter), subtle gradients, and intuitive navigation. Minimalist but polished.
